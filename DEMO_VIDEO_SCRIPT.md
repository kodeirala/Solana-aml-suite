## 🎥 Video Production Tips

### Visuals to Include:
1. Dashboard walkthrough (50% of video)
2. Code snippets (10%)
3. Architecture diagrams (10%)
4. Statistics and impact slides (20%)
5. Opening/closing cards (10%)

### Recording Setup:
- Use Loom or OBS Studio
- 1080p resolution minimum
- Clear audio (use good mic)
- Keep mouse movements smooth
- Practice the demo 3-4 times first

### Editing:
- Add background music (low volume)
- Use transitions between sections
- Add text overlays for key points
- Keep it punchy - cut dead air
- Export at 1080p 60fps

### Thumbnail Ideas:
- "From $4.3B Fines to Zero"
- "Real-time AML for Solana"
- "Stop Money Laundering On-Chain"
- Dashboard screenshot with "BLOCKED" overlay

### Call to Action:
- Link to GitHub in description
- Link to live demo
- Link to bounty submission
- Encourage stars/forks

## 🎬 Recording Checklist

Before hitting record:
- [ ] Dashboard is running and populated with data
- [ ] Practice demo 3 times
- [ ] Script printed out
- [ ] Good lighting on your face (if showing face)
- [ ] Microphone tested
- [ ] Close unnecessary tabs/apps
- [ ] Set browser to fullscreen
- [ ] Disable notifications
- [ ] Have backup recording software ready

During recording:
- [ ] Speak clearly and confidently
- [ ] Maintain energy throughout
- [ ] Follow script but sound natural
- [ ] Show real features, no faking
- [ ] Keep mouse movements deliberate

After recording:
- [ ] Review full video
- [ ] Check audio quality
- [ ] Edit out mistakes
- [ ] Add music/transitions
- [ ] Export at high quality
- [ ] Upload to YouTube
- [ ] Share on Twitter/LinkedIn
